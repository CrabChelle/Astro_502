The excel spreadsheets came from the EXOFOP database and contains information about the stars. The filtered one only has the data that covered the relevant wavelengths (H-alpha, Lithium I, and Calcium 2 H and K) at an SNR of at least 70. 
Multidownload TRES is used for data taken with TRES, FIES for data with FIES, etc
Currently there is no way to automatically download the data to your computer, will likely have to be slightly altered at some point.
